# Launch Plan: Packages, ROI, and Onboarding

## Packaging Summary
- Core: CRM + basic HR sync + standard dashboards.
- Pro: automation workflows + advanced analytics + API access.
- Enterprise: SSO + audit log + white-label + compliance pack + priority support.

## ROI Materials
- Standard KPI pack:
  - Pipeline velocity
  - Lead response time
  - Task SLA compliance
  - Win rate and cycle time
- "Before/After" templates for customer success reviews.

## Onboarding Offer
- Paid onboarding (fixed price):
  - Data import and field mapping
  - Pipeline setup and team roles
  - Automation rules (SLA tasks)
  - Training session + Q&A
- Optional migration support for historical CRM/HR data.

## Sales Motion
- 30-day pilot with defined success criteria.
- Convert to annual contract for Pro/Enterprise.
- Standard contract: 12-24 months for Enterprise tiers.

## Timeline (Example 12 Weeks)
1. Week 1-2: Pricing and packaging finalization.
2. Week 3-4: Enterprise feature availability.
3. Week 5-8: Integrations rollout.
4. Week 9-10: Ops/SLA readiness.
5. Week 11-12: Pilot and launch.

## Metrics to Track
- New MRR per month
- Conversion rate from pilot
- Average seats per customer
- Churn and expansion MRR
